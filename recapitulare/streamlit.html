Streamlit — Recapitulare

Framework web Python fără HTML/CSS/JS · Toate conceptele pentru examen

Introducere & Pornire

Streamlit este un framework web Python care permite crearea de aplicații web fără HTML/CSS/JS. Aplicația este un fișier .py rulat de un server.

# Import
import streamlit as st   # convenție: alias 'st'

# Configurare pagină (TREBUIE să fie primul apel st.*)
st.set_page_config(
    page_title="Aplicația Mea",
    layout="wide"       # sau "centered" (implicit)
)
Pornire aplicație: streamlit run app.py — NU python app.py, NU streamlit start.
URL implicit: http://localhost:8501

⚠️ Modelul de Execuție TOP-DOWN

La ORICE interacțiune a utilizatorului (click buton, mișcare slider, text input), Streamlit re-execută INTEGRAL scriptul de sus în jos.

  • Variabilele simple se pierd la fiecare re-execuție
  • Ordinea codului = ordinea afișării elementelor
  • Nu există o "buclă web" manuală
  • Starea trebuie salvată în st.session_state
La apăsarea st.button("OK"): întreg fișierul Python se re-execută de la început (nu doar blocul din if). Aplicația nu se închide.

Funcții de Afișare

st.title("Titlu Principal")       # cel mai mare
st.header("Header")
st.subheader("Subheader")
st.write("Text sau date")         # universal — afișează orice
st.markdown("**bold** _italic_")  # suportă Markdown
st.caption("Text mic")
st.text("Text monofonts")

st.code("x = 1 + 2", language="python")  # cod cu sintax highlighting
st.latex(r"\int_a^b f(x) dx")            # formule LaTeX

st.metric(label="Vânzări", value="1,234", delta="+5%")  # indicator cu variație
st.json({"key": "value"})               # obiect JSON

Afișare Tabele

st.dataframe — INTERACTIV

Scroll, sortare pe coloane. Acceptă DataFrame, liste de dicționare, etc.

st.dataframe(df)

st.table — STATIC

Tabel fix, fără interactivitate.

st.table(df)

st.data_editor — EDITABIL

Utilizatorul poate edita valorile din tabel.

st.data_editor(df)
st.table ≠ singurul mod de a afișa DataFrame. st.dataframe = interactiv, st.data_editor = editabil, st.write(df) funcționează de asemenea.

Widget-uri Input

Widget-urile de input returnează valori Python și controlează logica aplicației. NU sunt funcții de afișare. 
# Butoane și text
if st.button("Click me"):          # returnează True la click
    st.write("Buton apăsat!")

name = st.text_input("Nume:")       # returnează string
bio = st.text_area("Biografie:")    # text multi-linie

# Numeric
age = st.number_input("Vârstă:")
age = st.number_input("Vârstă:", min_value=0, max_value=120, value=25)
level = st.slider("Level:", 1, 10, 5)  # min, max, default

# Selecție
opt = st.selectbox("Alege:", ["A", "B", "C"])     # O singură opțiune
opts = st.multiselect("Alege:", ["A", "B", "C"])  # Mai multe opțiuni → returnează LISTĂ
gen = st.radio("Gen:", ["M", "F"])                  # Radio buttons
ok = st.checkbox("Accept termenii")               # True/False

# Dată
data = st.date_input("Data:")
fisier = st.file_uploader("Încarcă fișier:")
st.multiselect fără nicio selecție returnează o listă goală [], nu None.

Sidebar & Coloane

# Sidebar — bara laterală
st.sidebar.write("În sidebar")
section = st.sidebar.radio("Navigare:", ["Home", "Settings", "Result"])
with st.sidebar:
    st.title("Meniu")

# Coloane
col1, col2 = st.columns(2)
with col1:
    st.write("Coloana stânga")
with col2:
    st.write("Coloana dreapta")

# Coloane cu proporții diferite
col1, col2, col3 = st.columns([2, 1, 1])
st.slider = widget de INPUT, NU de layout! Elementele de layout sunt: sidebar, columns, expander, tabs.

Tabs & Expander

# Tabs

    st.write("Conținut Tab 1")

# Expander — secțiune colapsabilă
with st.expander("Detalii avansate"):
    st.write("Conținut ascuns implicit")

⚠️ Session State

st.session_state este un dicționar care păstrează valori între re-execuțiile scriptului.

Comportament

AfirmațieAdevărat?
Este un dicționar ce păstrează valori între re-execuții✅ ADEVĂRAT
Poate memora numărul de click-uri✅ ADEVĂRAT
Poate reține obiecte (ex. DataFrame)✅ ADEVĂRAT
Se golește automat la fiecare apăsare de buton❌ FALS — NU se golește automat
# Inițializare
if "counter" not in st.session_state:
    st.session_state.counter = 0

# Modificare cu callback
def increment():
    st.session_state.counter += 1

st.button("Click", on_click=increment)
st.write(f"Clicks: {st.session_state.counter}")

# Variabilele simple se pierd la re-execuție!
count = 0        # se resetează la 0 de fiecare dată
count += 1       # NU funcționează ca numărător persistent

Callbacks

# on_click — buton
def on_click_fn():
    st.session_state.value = "schimbat"

st.button("OK", on_click=on_click_fn)

# on_click cu argumente
def adauga(n):
    st.session_state.total += n

st.button("Adaugă 5", on_click=adauga, args=(5,))

# on_change — selectbox, text_input etc.
st.selectbox("Opt:", opts, on_change=callback, key="sel")
