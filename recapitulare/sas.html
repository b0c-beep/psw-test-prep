SAS Programare — Recapitulare

DATA step, PROC step, formate, combinare seturi de date

Sintaxă de bază

REGULA #1: Fiecare declarație se termină cu PUNCT ȘI VIRGULĂ (;) 
/* Structura unui program SAS */
libname mylib '/home/user/date';   /* declară o bibliotecă */

/* DATA step */
data mylib.studenti;
    set mylib.studenti_raw;
    /* instrucțiuni */
run;

/* PROC step */
proc print data=mylib.studenti;
run;

/* Comentarii: /* */ sau * text ; */
* Acesta este un comentariu;
/* Acesta de asemenea */
Programul SAS are 2 tipuri de secțiuni: DATA step (pregătire date) și PROC step (rapoarte, analize). LIBNAME nu este o secțiune.

Tipuri de date & Variabile

NUMERIC

  • Implicite dacă nu există $
  • Valoare lipsă = . (punct)
  • Lungime implicită = 8 bytes

CARACTER

  • Marcate cu $ în INPUT
  • Valoare lipsă = spații
  • Orice conținut mixt → CARACTER
O coloană cu valori mixte (numere + text, ex: "3456_1", "Romano", "Choi") → tip CHARACTER 
/* Citire cu INPUT */
input Nume $ Pret Categorie $;
/* $ = variabilă caracter; fără $ = numeric */

/* Lungime variabilă nouă (EVITAREA TRUNCHIERII) */
length Raportare $ 8;   /* ÎNAINTE de utilizare! */
if Tip = 'A' then Raportare = 'Acceptat';
else Raportare = 'Respins';
Trunchierea: SAS fixează lățimea variabilei noi după PRIMA observație. Dacă prima valoare e "Mic" (3 car.) și următoarea e "Mediu" (5 car.), "Mediu" va fi trunchiat! → Folosiți LENGTH ÎNAINTE.

⚠️ LIBNAME vs INFILE

LIBNAME → DIRECTOR

libname prod '/home/user';
/* pointer la FOLDER */
/* permite prod.numeset */

INFILE → FIȘIER

infile '/home/user/date.txt';
/* pointer la FIȘIER */
/* pentru citire date raw */
libname date '/home/user/fisier.txt'; → EROARE (libname pe fișier, nu director)
infile '/home/user/'; → EROARE (infile pe director, nu fișier) 
/* Corect */
libname date '/home/user';          /* director ✓ */
infile '/home/user/date.txt';        /* fișier .txt ✓ */

/* WORK = bibliotecă temporară */
data work.temp;  /* sau */  data temp;  /* ambele sunt WORK */
/* Fișier temporar = UN singur nivel (temp), nu mylib.temp */
/* Sters la închiderea sesiunii SAS */

DATA Step — Citire date

/* Delimitare prin spațiu (implicit) */
data produse;
    infile '/home/user/produse.txt';
    input Nume $ Pret Categorie $;
run;

/* Delimitare prin virgulă — DSD */
data produse;
    infile '/home/user/produse.csv' DSD;
    input Nume $ Pret Categorie $;
run;
/* DSD: delimitator = virgulă; 2 virgule = valoare lipsă; ghilimele pentru câmpuri */

/* Alt delimitator */
infile '...' DLM='*';   /* DLM = DELIMITER */

/* DATALINES — date în cod */
data test;
    input Cod Nota;
datalines;
1 8
2 7
3 9
;

/* Column input (lățime fixă) */
input Nume $ 1-8  Pret 9-11  Cat $ 12-19;
/* Poate citi în orice ordine; valori lipsă = spații libere */

/* Formatted input */
input @1 Nume $8. @9 Pret dollar4. @13 Cat $8.;
/* @ = poziția de start; format la final */

/* @@ — mai multe înregistrări pe aceeași linie */
input Varsta @@;

data total;

proc sort data=a; by ID; run;

/* PROC APPEND — adaugă b la a (nu recitește a) */
proc append base=myset data=new_obs; run;

⚠️ IN= — Variabile de corespondență

data rezultat;
    merge a(in=inA) b(in=inB);
    by ID;
    /* inA=1 dacă ID există în a; inB=1 dacă ID există în b */
run;
CondițieEchivalent SQLRezultat
if inA and inB;INNER JOINDoar ID-urile comune
if inA and not inB;LEFT ANTI JOINDoar în A, nu și în B
if not inA and inB;RIGHT ANTI JOINDoar în B, nu și în A
if inA or inB;FULL OUTER JOINToate (același cu fără IF)
IN= creează variabile TEMPORARE (0 sau 1) care nu apar în setul de date de ieșire.

IF / ELSE IF / SELECT

/* IF-THEN */
if Varsta lt 20 then Grupa = 1;
else if Varsta lt 40 then Grupa = 2;
else Grupa = 3;
/* ⚠️ ELSE IF este esențial — IF-uri independente ar suprascrie greșit */

/* IF cu grup DO */
if Nota >= 5 then do;
    Status = 'Promovat';
    Bonus = 100;
end;

/* IF fără THEN = subset (filtrare) */
if Sex eq 'F';   /* păstrează DOAR rândurile unde Sex='F' */

/* DELETE */
if missing(Varsta) then delete;   /* nu adaugă rândul */

/* SELECT */
select(Categorie);
    when ('legume', 'fructe') Reducere = 0.25;
    when ('cosmetice') Reducere = 0.30;
    otherwise Reducere = 0;
end;
/* Fără OTHERWISE și toate false → EROARE! */

⚠️ WHERE vs IF

CaracteristicăWHEREIF (fără THEN)
Disponibil în PROC✅ DA❌ NU
Variabile create în DATA step❌ NU (doar din setul de intrare)✅ DA
Operatori specialiIS MISSING, BETWEEN, CONTAINS, LIKEOperatori standard
/* WHERE operatori speciali */
where Varsta is missing;                  /* valoare lipsă */
where Nota between 5 and 10;            /* interval INCLUSIV */
where Adresa contains 'Str';             /* conține subșirul */
where Nume like 'A_e%';                  /* sablon: _ = un car., % = oricâte */
where Style in ('RANCH', 'SPLIT');       /* lista de valori */

/* Operatori comparație */
where Suma <= 5000 and (Cont='101' or Rata eq 0.095);
/* = EQ, ^= NE, < LT, <= LE, > GT, >= GE */

DO / WHILE / UNTIL / OUTPUT

/* DO cu contor */
do i = 1 to 5;
    ...
end;

do i = 1 to 10 by 2;   /* pas = 2: 1,3,5,7,9 */
do i = 10 to 1 by -1;  /* descrescător */

/* DO WHILE — verificare ANTERIOARĂ */
do while (i <= 10);
    ...  /* s-ar putea să nu se execute deloc */
end;

/* DO UNTIL — verificare POSTERIOARĂ */
do until (i > 10);
    ...  /* se execută MINIM O DATĂ */
end;
/* OUTPUT în buclă — scrie un rând la fiecare iterație */
data lunar;
    do luna = 1 to 12;
        Vanzari = Vanzari_medii * luna;
        output;   /* scrie rândul pentru fiecare lună */
    end;
run;

Acumulator SUM (sum statement)

/* var + expresie; (fără =) = acumulator */
Total + Venit;
/* - Inițializat automat cu 0 (nu cu .)
   - Reține valoarea de la o observație la alta
   - Ignoră valorile NULL (lipsă)
   - Rolul: sumă cumulată */
Total + Venit; ≠ Total = Total + Venit;. Prima este sum statement (acumulator persistent). A doua este o atribuire normală (pierde valoarea la rândul următor dacă nu e inițializată).

⚠️ Formate & PROC FORMAT

  • Formatul de afișare conține punct (.) și se aplică unei variabile
  • Afectează modul de afișare, NU datele stocate
  • Formatul de citire (informat) = folosit la INPUT
  • Formatul de afișare = folosit la FORMAT, PUT
/* Formate predefinite */
format Salary dollar10.2;         /* $108,255.00 */
format DataComanda date7.;          /* 11JAN24 */
format DataComanda mmddyy8.;        /* 01/11/24 */

/* PROC FORMAT — formate utilizator */
proc format;
    /* Format NUMERIC */
    value nota
        low-4.4 = 'Picat'
        4.5-high = 'Promovat';

    /* Format NUMERIC cu intervale */
    value nivel_punctaj
        low-<50 = 'Scazut'
        50-<80 = 'Mediu'
        80-high = 'Ridicat';

    /* Format CARACTER — $ prefix */
    value $gen
        'M' = 'Masculin'
        'F' = 'Feminin'
        other = 'Necunoscut';
run;

/* Aplicare format */
format Nota nota.;        /* punct obligatoriu la final! */
format Gen $gen.;         /* format caracter */

/* Formate persistente — salvate în bibliotecă */
libname formate '/home/user/fmts';
options fmtsearch=(formate);
proc format library=formate;
    value pl low-10='acceptabil' 11-high='peste limita';
run;
PROC FREQ numără valorile formatate. Dacă există format aplicat pe variabilă, frecvențele sunt grupate conform formatului.

⚠️ PUT vs INPUT

PUT → numeric la CARACTER

VarChar = PUT(VarNum, dollar10.2);
/* Returnează STRING */
/* Folosește format afișare */

INPUT → caracter la NUMERIC

VarNum = INPUT(VarChar, dollar10.2);
/* Returnează NUMĂR */
/* Folosește informat de citire */
PUT = numeric → caracter (afișare). INPUT = caracter → numeric (citire).
Examenul confundă frecvent cele două!

PROC SORT

/* Sortare simplă */
proc sort data=studenti;
    by Nota;
run;

/* Descrescător pe 2 variabile — descending se REPETĂ */
proc sort data=angajati;
    by descending Postal_Code descending Employee_ID;
run;

/* Salvare în set nou */
proc sort data=a out=b;
    by ID;
run;
proc sort data=studenti; run; fără BY → EROARE (BY este obligatoriu)

PROC MEANS

/* Statistici descriptive */
proc means data=studenti;
    var Nota1 Nota2;
run;

/* Cu grupuri (necesită SORT prealabil!) */
proc sort data=vanzari; by LunaComanda; run;
proc means data=vanzari mean;
    by LunaComanda;
    var ValoareVanzare;
run;

/* Salvare rezultate */
proc means noprint data=achizitii;
    by IDCumparator;
    var Valoare;
    output out=rezumat sum(Valoare)=Total mean(Valoare)=Medie;
run;
/* noprint = fără raport pe ecran; output out= = salvare */
/* range mean nonobs maxdec=1 = opțiuni statistici */

PROC FREQ

/* Frecvențe simple */
proc freq data=studenti;
    tables Sex;
run;

/* Cu format aplicat — numără valorile FORMATATE */
proc freq data=candidati;
    tables Punctaj / nocum;
    format Punctaj nivel_punctaj.;
run;
/* nocum = fără frecvențe cumulative */

/* Creare set de ieșire */
proc freq data=a;
    tables Categorie / out=frecvente;
run;
PROC FREQ numără atât variabile caracter cât și numerice. Numără valorile formatate dacă există format aplicat.

PROC PRINT & TITLE

proc print data=studenti;
    var Nume Nota;           /* afișează DOAR aceste variabile */
    where Sex eq 'F';         /* filtrare — WHERE valid în PROC */
    by Gen;                    /* necesită date sortate după Gen */
    label;                     /* afișează etichetele coloanelor */
run;

/* TITLE */
title1 "Raport principal";
title2 "Subtitlu";

title;   /* fără număr = RESETEAZĂ TOATE titlurile */
/* Rapoartele după title; NU au titlu */
title; (fără număr și fără text) RESETEAZĂ toate titlurile anterioare. Rapoartele după această linie nu vor mai afișa niciun titlu.

PROC SQL

proc sql;
    /* Creare tabel din join */
    create table rezultat as
    select * from proiecte p
    inner join echipe e on p.IDProiect = e.IDProiect;

    /* Full join */
    select * from a full join b on a.ID = b.ID;

    /* Left join + IS NULL = fără corespondent */
    select a.* from abonati a
    left join plati p on a.ID = p.ID
    where p.ID is null;   /* abonați fără plăți */

    /* Grupare */
    select Regiune, avg(Vanzari) from date group by Regiune;
quit;

⚠️ ARRAY

/* Definire array — acces la variabile cu indici */
array raspunsuri{6} Q1-Q6;
/* Crează referință la variabilele Q1, Q2, Q3, Q4, Q5, Q6 */

/* Iterare array */
do i = 1 to 6;
    if raspunsuri{i} = 0 then raspunsuri{i} = .;
end;
/* Înlocuiește 0 cu valoare lipsă pentru Q1-Q6 */

/* DIM — numărul elementelor */
do i = 1 to dim(raspunsuri);  /* = do i = 1 to 6; */

/* Array 2D */
array matrice{3, 4};   /* 3 rânduri, 4 coloane */

/* _temporary_ — nu creează variabile în setul de date */
array limite{3} _temporary_ (5 7 9);

Macro-uri

/* Variabilă macro */
%let an = 2024;
data studenti_&an;  /* & = referință la variabilă macro */

/* Macro program */
%macro raport(set, var);
    proc means data=&set;
        var &var;
    run;
%mend;
%raport(studenti, Nota);

/* CALL SYMPUT — creare variabilă macro din date */
call symput('maxim', Nota);

/* Variabile macro speciale */
&SYSDATE   /* data curentă */
&SYSDAY    /* ziua săptămânii */
