Liste

Listele sunt ordonate, modificabile și permit duplicate. Se definesc cu [].

ProprietateValoare
OrdonateDA
Modificabile (mutable)DA
Permit duplicateDA
Acces prin indexDA

Operații de bază

a = [1, 2, 3, "text", None, [], ()]
print(len(a))        # 7 — numără TOATE elementele, inclusiv None/[]/() 

a.append(99)        # adaugă UN element la sfârșit
a.insert(0, "x")    # inserează la poziția 0
a.remove(2)         # șterge prima apariție a valorii 2
del a[0]            # șterge elementul de la index 0
a.pop()             # șterge și returnează ultimul element
list.append() acceptă UN SINGUR argument. a.append(2, "Scott") → ERROR

Slicing (tăiere)

nums = [0, 1, 2, 3, 4, 5]
nums[1:4]    # [1, 2, 3] — capătul din dreapta EXCLUSIV
nums[:3]     # [0, 1, 2]
nums[3:]     # [3, 4, 5]
nums[-1]     # 5 — ultimul element
nums[::-1]   # [5, 4, 3, 2, 1, 0] — inversare

# Indici pari: range(0, len, 2)
t = (1, 2, 4, 3, 8, 9)
[t[i] for i in range(0, len(t), 2)]  # [1, 4, 8] — indici 0, 2, 4

List Comprehension

[x**2 for x in range(5)]               # [0, 1, 4, 9, 16]
[x for x in nums if x % 2 == 0]       # doar numerele pare

# Parsare input text cu comprehension:
names = [x.strip() for x in product_names.split(",")]
prices = [float(x.strip()) for x in prices_str.split(",")]
len([1, 2, 3, None, (), {}]) → 6. Elementele goale (None, tuple gol, dict gol) se numără ca elemente normale.

Tupluri

ProprietateValoare
OrdonateDA
Modificabile (mutable)NU — IMUTABILE
Permit duplicateDA
Sintaxă()
t = (1, 2, 3)
x, y, z = t         # unpacking
tuple([1, 2, 3])     # (1, 2, 3) — din listă

# Returnare multiplă din funcție = tuplu
def stats(x):
    return min(x), max(x), sum(x)  # returnează tuplu
Tuplul este IMUTABIL:
  • my_tuple.append(x) → AttributeError (tuplul nu are metoda append)
  • a = (1, 2); a[0] += 1 → TypeError
  • del a (unde a este tuplu) → funcționează — șterge variabila (nu elementul)

Seturi

ProprietateValoare
OrdonateNU — neordonate
Valori uniceDA — elimină duplicatele
Sintaxă{} sau set()
s = {1, 2, 3, 3, 2, 1}   # {1, 2, 3} — duplicatele sunt eliminate

# Operații pe seturi
s1 | s2    # reuniune (union)
s1 & s2    # intersecție
s1 - s2    # diferență

# Operatorul 'in' funcționează cu seturi
2 in {1, 2, 3}  # True

Dictionare

ProprietateValoare
AccesPRIN CHEIE (nu prin index numeric)
CheiUnice + Imutabile (string, int, tuple)
ValoriOrice tip, poate fi duplicat
Sintaxă{"cheie": valoare}
d = {"john": 40, "peter": 45}

d["john"]              # 40 — acces prin cheie
d["city"] = "NY"      # adaugă/modifică
d.get("C", 0)         # 0 — returnează default dacă cheia nu există
d.items()             # iterare perechi (cheie, valoare)
d.keys(), d.values() # cheile / valorile

# Comparare dicționare
d1 = {"john": 40}; d2 = {"john": 466}
d1 == d2   # False — compară conținutul (valorile diferă)

# Operatorul 'in' verifică CHEILE
"bar" in {"foo": 1, "bar": 2}   # True

# Exemplu: dict.get + dict acces
date = {"A": 10, "B": 20}
date.get("C", 0) + date["A"]   # 0 + 10 = 10
Operatorul in funcționează cu list, set și dict (pentru dict verifică cheile).

If / Elif / Else

if x > 0:
    print("pozitiv")
elif x == 0:
    print("zero")
else:
    print("negativ")

# Operator ternar (ternary)
result = "Positive" if x > 0 else "Negative"
Python este CASE-SENSITIVE: 'Amir' ≠ 'amir'.
Exemplu: if 'amir' in ['Amir', 'Barry'] → False → merge pe else.
Blocul se termină printr-o linie indentată mai puțin decât linia anterioară (spre deosebire de {} în alte limbaje).

Bucle For / While

# For — iterare secvență
for x in [10, 20]:
    for y in ["Chair", "Table"]:
        print(x, y)
# 10 Chair / 10 Table / 20 Chair / 20 Table (produs cartezian)

# While
x = 0
while x < 100:
    x += 2
print(x)   # 100

# break / continue / pass
break      # iese complet din buclă
continue   # sare la iterația următoare
pass       # placeholder, nu face nimic
La bucle imbricate, numărați iterațiile: range(10) × range(-1,-10,-1) = 10 × 9 = 90 iterații.

range()

range(5)           # 0, 1, 2, 3, 4
range(2, 8)        # 2, 3, 4, 5, 6, 7  (capăt EXCLUSIV)
range(5, 16, 5)   # 5, 10, 15
range(0, len(t), 2)  # indici pari: 0, 2, 4, ...
range(-1, -10, -1)  # -1, -2, ..., -9  (9 valori)
range() acceptă doar INT. range(0.5, 5.5, 0.5) → TypeError

Funcții și Scope

# Parametri cu valoare implicită
def power(base, exponent=2):
    return base ** exponent

power(3)      # 9 (exponent=2 implicit)
power(3, 3)   # 27 (exponent=3 suprascrie)

# *args și **kwargs
def f(*args, **kwargs): ...  # liste și dicționare de argumente

# Returnare multiplă (returnează tuplu)
def stats(x):
    return min(x), max(x), sum(x)
SCOPE LOCAL vs GLOBAL:
O variabilă definită în interiorul unei funcții este LOCALĂ — nu modifică variabila globală cu același nume. 
salary = 8000                    # variabilă GLOBALĂ
def printSalary():
    salary = 12000               # variabilă LOCALĂ (diferită!)
    print(salary)                # 12000

printSalary()
print(salary)                    # 8000 — variabila globală e neschimbată

# Modificare globală — folosiți 'global'
def modify():
    global salary
    salary = 12000               # acum modifică globala
Liste vs scalari la pasare în funcții:
Listele sunt mutable → modificările în funcție afectează originalul. 
def f(value, values):
    values[0] = 44       # modifică lista originală (mutable)
    value = 99            # NU modifică variabila originală (scalar)

t = 3; v = [1, 2]
f(t, v)
print(t, v[0])   # 3 44 — t neschimbat, v[0] modificat

Lambda, Map, Filter

# Lambda — funcție anonimă
square = lambda x: x**2
square(5)   # 25

# map — aplică funcția pe fiecare element
list(map(lambda x: x*2, [1, 2, 3]))   # [2, 4, 6]

# filter — filtrează elementele
list(filter(lambda x: x%2==0, [1,2,3,4]))  # [2, 4]

# sorted cu key
students = [("Ana", 8), ("Bob", 6), ("Cara", 9)]
sorted(students, key=lambda s: s[1])  # sortare după notă

Sortare

sorted() — listă nouă

note = [7,9,6,10]
rez = sorted(note)
# note = [7,9,6,10] NESCHIMBAT
# rez  = [6,7,9,10] NOUĂ

.sort() — modifică in-place

note = [7,9,6,10]
note.sort()
# note = [6,7,9,10] MODIFICAT
# returnează None !
Examen: note=[7,9,6,10]; rez=sorted(note); print(note[0], rez[0]) → afișează 7 6 (note[0] rămâne 7, rez[0]=6 ca minim)

Șiruri de caractere

s = "Seminar, pachete software"

s.strip()       # elimină spațiile de la capete

names = ['Amir', 'Barry', 'Charles', 'Dao']
names[-1][-1]   # 'o' — ultimul caracter al ultimului element ('Dao')

x = "10"; y = 5
x + str(y)    # "105" (str trebuie să fie un ȘABLON, nu clasa str!)
x + str       # TypeError — str este o CLASĂ, nu un șir

Citire / Scriere fișiere

# Citire cu context manager
with open("raport.txt", "r") as f:
    linii = f.readlines()    # listă de linii
    print(len(linii))         # NUMĂRUL DE LINII din fișier

# Moduri de deschidere
# "r" = citire, "w" = scriere (suprascrie), "a" = append, "rb" = binar

# Tratare excepții
try:
    with open("f.txt") as f: ...
except (ValueError, TypeError) as e:
    print(e)
finally:
    ...   # se execută ÎNTOTDEAUNA (chiar și la excepție)
