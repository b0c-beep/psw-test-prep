⚠️ loc vs iloc — cea mai importantă diferență

df.loc — ETICHETE

  • Accesează după etichete (index names)
  • Slicing INCLUSIV la ambele capete
  • Acceptă mști booleene
  • Acceptă etichete string

df.iloc — POZIȚII

  • Accesează după poziție întreagă (0-based)
  • Slicing capăt EXCLUSIV
  • NU acceptă mști booleene
  • NU acceptă etichete string → EROARE
  • Acceptă indici negativi (-1 = ultimul)
# DataFrame cu index [1,2,3,4,5]
df.loc[3]      # rândul cu ETICHETA 3 (al treilea rând dacă index=1,2,3,4,5)
df.iloc[3]     # rândul de pe POZIȚIA 3 (al PATRULEA rând, 0-indexed)
# ⚠️ Cele două POT FI DIFERITE dacă indexul nu e 0,1,2,...

# Exemple concrete
df.iloc[1:3, 0:2]   # rânduri 1,2 (exclusiv 3) și coloane 0,1 (exclusiv 2)
df.iloc[:, 0:3]    # PRIMELE 3 coloane (pozițiile 0, 1, 2)
df.iloc[-1]        # ULTIMUL rând

# loc cu mască booleană
df.loc[df['Pret'] > 100]              # ✓ loc acceptă mști booleene
df.loc[(df['A']>0) & (df['B']>0)]   # ✓ & pentru AND
df.loc[(df['A']>0) | (df['B']>0)]   # ✓ | pentru OR

# Modificare cu loc
df.loc[df['Pret'] >= 100, "CategoriePret"] = "mare"
# Modifică coloana CategoriePret DOAR pentru rândurile cu Pret>=100
df.iloc['banana'] → EROARE (iloc are nevoie de index INTEGER, nu string).
df.iloc[df['col']>0] → EROARE (iloc nu acceptă condiții booleene, folosiți loc).
set_index cu inplace=False:
data.set_index("last_name", inplace=False) — returnează un DataFrame NOU dar nu modifică data. Deci data.loc[['Andrade']] după această linie → EROARE (indexul e tot cel implicit).

Valori Lipsă (NaN)

# Identificare
df.isnull().sum()          # câte NaN pe fiecare coloană

# Completare (fillna)
df['Pret'].fillna(df['Pret'].mean())       # cu media
df['Pret'] = df['Pret'].fillna(df['Pret'].mean())  # salvare modificare
df.fillna(df.mean())        # completare cu media pe toate coloanele numerice
df.fillna(method='ffill')   # forward fill (propagă valoarea anterioară)
df.fillna(method='bfill')   # backward fill

# Eliminare
df.dropna()             # elimină rândurile cu orice NaN
df.dropna(subset=['A']) # elimină rândurile cu NaN în coloana A

# Interpolate
df.interpolate()        # interpolare liniară

# lambda apply pentru valori lipsă condiționate
df['Gender'] = df.apply(
    lambda row: 'Female' if row['Name']=='Alice' and pd.isnull(row['Gender'])
               else row['Gender'], axis=1)
# Completează Gender cu 'Female' DOAR dacă Name='Alice' ȘI Gender e NaN

Merge / Concat

# MERGE — joncțiune (implicit INNER)
pd.merge(df1, df2, on='Date')                  # INNER — doar valorile comune
df1.merge(df2, on='Date', how='left')           # LEFT — toate din stânga + NaN
pd.merge(df1, df2, on='ID', how='right')       # RIGHT
pd.merge(df1, df2, on='ID', how='outer')       # OUTER — reuniunea

# Indicator — coloana _merge
pd.merge(df1, df2, how='outer', indicator=True)
# Coloana _merge are valorile: 'left_only', 'right_only', 'both'

# Coloane cu același nume → sufixe _x (stânga), _y (dreapta)

# CONCAT — concatenare VERTICALĂ (unul sub altul)
pd.concat([df1, df2])          # concatenare verticală
pd.concat([df1, df2], axis=1)  # concatenare orizontală
merge implicit = INNER — păstrează doar rândurile cu valori comune.
concat = verticală (unul sub altul), nu joncțiune.

Indicator merge — Left join + IS NULL (corespondent fără plăți)

# Abonați FĂRĂ nicio plată (echivalent SQL left join + where null)
merged = pd.merge(abonati, plati, on='CodAbonat', how='left', indicator=True)
fara_plati = merged[merged['_merge'] == 'left_only']

Apply / Map / Drop

# apply — aplică funcție pe coloană sau rând
df['col'].apply(lambda x: x * 0.9)   # pe coloană (scalar)
df.apply(func, axis=1)               # pe fiecare rând (axis=1)

# map — recodificare Series
df['col'].map(lambda x: 'High' if x > 400 else 'Low')

# drop — ștergere coloane sau rânduri
df.drop(['col1', 'col2'], axis=1)            # axis=1 = COLOANE
df.drop(['col1', 'col2'], axis=1, inplace=True)  # modificare in-place
df.drop([0, 1], axis=0)                       # axis=0 = RÂNDURI

# rank — atribuire ranguri
df['Nota'].rank(ascending=False)  # egalitati -> medie (ex 4.5)
axis=1 = coloane | axis=0 = rânduri. La drop, axis=1 înseamnă că ștergem coloane.

Preprocesare — Ordinea pipeline

  1. Valori lipsă (fillna / dropna / interpolate)
  2. Codificare non-numeric (Label Encoding, One-Hot, Target)
  3. Outlieri (IQR = Q3-Q1; limite: Q1-1.5*IQR / Q3+1.5*IQR)
  4. Scalare (StandardScaler, MinMaxScaler, RobustScaler)
